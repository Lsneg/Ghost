    Формы

    Input

    Самый обычный <input> :


    1. <form>
    2.   <label for="inp-example">Имя лейбла</label>

    3.   <input type="text" id="inp-example">
    4. </form>

    Checkbox

    Чекбокс необходимо использовать внутри тега <label class="checkbox"> с описанием значения.

    Обязательно используйте class="checkbox" при выстраивании чекбоксов в линию!
    В других случаях применять класс не обязательно.     
    1. <label class="checkbox">

    2.   <input type="checkbox"> Имя лейбла

    3. </label>

    Radio

    Как и чекбоксы, радио-кнопки необходимо помещать в <label>, который в свою очередь должен иметь класс radio. Чтобы определить один из елементов выбраным по умолчанию, необходимо ему добавить свойство <checked>. 

    1. <label class="radio">

    2.   <input type="radio" name="radio" checked>Левша

    3. </label>

    4. <label class="radio">

    5.   <input type="radio" name="radio">Правша

    6. </label>

    Textarea

    Высоту тектового поля можна задавать с помошью атрибута rows.

    1. <textarea rows="3"></textarea>

    Select

    С выпадающим списком:

    1. <select>

    2.   <option>Мужик</option>

    3.   <option>Баба</option>

    4.   <option>Таиланд</option>

    5.   <option>Не определено</option>

    6. </select>

    Для возможности выбора нескольких элементов необходимо добавить свойство "multiple" 

    1. <select multiple="multiple">

    2.   <option>Мужик</option>

    3.   <option>Баба</option>

    4.   <option>Таиланд</option>

    5.   <option>Не определено</option>

    6. </select>
