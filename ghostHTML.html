HTML & CSS

Начало работы

Для начала необходимо подключить стили:

<!DOCTYPE html>
<html>
	<head>
		<link href="ghost.css" rel="stylesheet">
	</head>
</html>

Гост изначально разрабатывался для создания адаптивных сайтов, поэтому основными единицами измерения являются относительные. Для наилучшего результата рекомендуется вести разработку с шириной страницы в 1280px.

Сетка

Для начала необходимо создать контейнер <div class="block-group">, который будет содержать группу блоков.

По умолчанию имеет отступ сверху, слева и справа

Для добавления нижнего отступа необходимо присвоить класс .plate. Чтобы убрать верхний и нижний оступы — .slim

Внутренние отступы по умолчанию изменяются в зависимости от ширины страницы

Внутри контейнера необходимо создать блоки <div class="block">, которыми можно сформировать сетку. Для этого каждому блоку необходимо присвоить класс, указывающий ширину, например блок <div class="block w20"> будет иметь ширину 20%, "w30" — 30%, а "w50" соответственно 50% от ширины контейнера.

w20
w30
w50

Чтобы содержимое соседних блоков не слипалось необходимо вложить дополнительный элемент с классом .inblock, который по умолчанию имеет верхний и боковые отступы: 

<div class="block w20">
	<div class="inblock">w20</div>
</div>

При необходимости адаптировать количество элементов по ширине при разных разришениях экранов, необходимо добавить контейнеру класс <div class="block-group adapt">

Ширина страницы при разработке адаптивной сетки — 1280px. Поэтому число в классе, например w20 соответствует реальному числу ширины 20% только при просмотре на странице с шириной 1280px.

Input

Самый обычный <input> :


<form>
	<label for="inp-example">Имя лейбла</label><br>
	<input type="text" id="inp-example">
</form>

Checkbox

Чекбокс необходимо использовать внутри тега <label class="checkbox"> с описанием значения.

<label class="checkbox">
	<input type="checkbox">Поставить галочку
</label>
Обязательно используйте class="checkbox" при выстраивании чекбоксов в линию!
В других случаях применять класс не обязательно.

Radio

Как и чекбоксы, радио-кнопки необходимо помещать в <label>, который в свою очередь должен иметь класс radio. Чтобы определить один из елементов выбраным по умолчанию, необходимо ему добавить свойство <checked>. 

<label class="radio">
	<input type="radio" name="radio" checked>Левша
</label>
<label class="radio">
	<input type="radio" name="radio">Правша
</label>

Textarea

Высоту тектового поля можна задавать с помошью атрибута rows.

<textarea rows="3"></textarea>

Select

С выпадающим списком:

<select>
	<option>Мужик</option>
	<option>Баба</option>
	<option>Таиланд</option>
	<option>Не определено</option>
</select>

Для возможности выбора нескольких элементов необходимо добавить свойство "multiple" 

<select multiple>
